デビューのルーツ


四つ葉ココちゃん は中国・湖南省生まれの音楽アーティストであり、 海洋素食者 、平和主義者。

丨2021年頃、「 Fish_Lucky（ 錦鯉ちゃん / ラッキーちゃん ） 」名義のクリエイターとしてインターネットで Minecraft 関連動画を公開。
丨2022年9月30日、「 白梓TeAm工作室 」を設立。視聴者の心緒に寄り添う“サービス系”のコメディ動画を制作。 その後、 「 白梓TeAm 」 は度重なる小騒動や紛争を経て関心が薄れ、計画していた社名「白汐文化传播有限公司」も先に商標取得される。
丨2023年初頭、白梓TeAmは解散。“自分だけの情緒”を探す旅に入る。
丨2023年10月、重度うつ病と強迫症の診断を受け、その後 双極性障害へ移行。
丨2024年07月27日、初の楽曲『浮生』をNetEase Musicで非独占配信。ここで初めて 四つ葉ココちゃん 名義を使用。


「境界のない創作とルールの中での極致追求。—— たぶん、浮遊している方がいい。」

— そう四葉は考えている。



2025年07月14日、J-Popアルバム『彩海（さいかい）』で独立デビュー。編曲・作曲・作詞・演奏・歌唱・マスタリング・ミキシング・配信・ビジネス・SNS・ビジュアル・編集まで、すべてを単独で担う。

アジア太平洋地域の音楽伝統と現代文化を広めながら、世界の音楽文化と交流することを目指し、四葉は自国の理念を実践し、世界平和と世界文化の発展に貢献したいと考えています。



芸名の由来


四つ葉ココちゃん の後半「ココちゃん」は、TVアニメ『ご注文はうさぎですか？』の主人公「 ココア / Cocoa 」（ 保登 心愛 / ほと ここあ - Hoto Kokoa ）に由来する名前の変形です。
ホットチョコレートが好きなことから、四葉は自分もチョコレートを好むことを思い出し、四つ葉という名前には幸運の象徴としての意味を込めています。
四つ葉の四枚の葉は、それぞれ彼女自身の両性特徴（一般的には両性人格 / Androgyny）や双極性障害の特性を表しており、音楽制作においてもこの両性性と双極性は四葉に大きな助けとなっています。
また、視覚的にバランスが良く美しいため、「四つ葉ココちゃん」の文字は全て保持し、省略せずに使用しています。

多言語の芸名

· 四葉可可醬 / 四叶可可酱（中文艺名）：Bilibili の ACG 文化に由来し、「可可」はチョコレート製作用のココアパウダーから来ています。
· Yotsuba Choco（English Artist Name）：四つ葉ココちゃんのローマ字表記を変形させたもので、Choco は Chocolate の略です。
· 요츠바 코코쨩（한국 아티스트 이름）：日本語の発音をハングルに転写したものです。
· Cacao Trèfle Phase 4（Nom d’artiste français）：Trèfle はフランス語で「三つ葉」を意味し、Cacao Trèfle Phase 4 は「三葉可可 第四段階」の意味です。また、Phase 4 には「第四相」という意味があり、楽曲の実験性を象徴しています。

音楽スタイル


四つ葉ココちゃん は感情系音楽アーティストで、楽曲は複雑な感情や深い思考、社会的テーマを反映することが特徴です。
複数の楽器が異なる感情を同時に表現することで、個人の心情の交錯や葛藤を描き出しており、これを音楽の魂の一つと捉えています。

主な音楽ジャンル：J-POP / 心象ロック / 実験音楽 / エモーショナルロック

歌唱レンジ：A2〜A5（極限時）

楽器：鍵盤楽器（ Arturia KeyLab / AKAI MPK / M-VAVE ）、ベース「絹光孔雀」（Bacchus WJB-330 中国製）、エレキギター（Fender）

楽曲構成：四葉の楽曲は非常に独自的で、「ミライまど（Future Window）」では、前奏 - サビ - 主歌A - 主歌B - 主歌C - 独立サビ / 楽器共演ソロ - 主歌C - 予サビ（前奏変形） - 完全サビ - サックスソロ - シンセソロ - 弦楽ソロ - 主歌C といった複雑な構造を採用し、尾奏には三回の独立ソロパートで締めています。


「構造は物語を表現するものであり、物語が多様に展開する時、構造も変化すべきである」

— Future Window（Note）

孤島アーティスト（2025）概要

孤島アーティスト（2025） は、四つ葉ココちゃん が2025年に独立して発表した概念的年度アルバムです。

アルバムのテーマは 心理・社会・人文・感情 を軸に展開され、実存主義 と 不条理主義、そして個人的な 楽観的堕落主義 を表現しています。 これはアルバムの魂の一つであり、存在と心理状態の深い探究でもあります。

孤島アーティスト では、四つ葉ココちゃん が概念的な編曲により、各曲が独自の世界観を持つよう設計されています。様々な曲風を融合し、一見孤島の世界であっても、煌めく孤島の魂の都市を形成しています。
